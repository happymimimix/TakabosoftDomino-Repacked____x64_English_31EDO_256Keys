MIDI music editing software "Domino" owner's manual

This document is Domino's instruction manual. I think that you should read in order from the top for the time being.
Since it is a document to explain how to use the software to the last, I do not write MIDI specifications, how to compose, etc. For those parts, please study at other websites.

In addition, it is difficult to explain all the functions, so it is a bit overwhelming, but if there are any unclear points etc, please ask with a bulletin board etc.
